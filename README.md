# Single-cell-multi-omics-integration

Single-cell multi-omics data analysis pipeline for integrating subclone structures from paired genome and transcriptome data in primary gastric and metastatic colon
cancers. 


## Reference
[Single cell multi-omic mapping of subclonal architecture and pathway phenotype in primary gastric and metastatic colon cancers](https://www.biorxiv.org/content/10.1101/2022.07.03.498616v1)
